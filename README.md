# Feedback Form Projec

Тестовое задание

## Стек технологий
- **Frontend**: Vanilla JS (ES6+), CSS3 (Variables, Grid/Flex), HTML5.
- **Backend**: Node.js + Express.
- **Интеграции**: DaData (подсказки ФИО), Yandex SmartCaptcha.

## Особенности реализации
- **Phone Mask**: Самописная реализация на чистом JS (без библиотек).
- **Security**: Валидация данных на фронтенде и бэкенде + проверка капчи на стороне сервера.
- **Performance**: Использование Debounce для запросов к API подсказок.
- **UX/UI**: Современный Dark UI, адаптивность, состояния загрузки и обработки ошибок.

## Запуск проекта
1. Склонируйте репозиторий.
2. Выполните команду `npm install` для установки зависимостей.
3. Создайте файл `.env` в корне и добавьте свои ключи.
4. Запустите сервер: `node server.js`.
5. Откройте `http://localhost:3000`.
